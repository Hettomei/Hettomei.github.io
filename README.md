Hettomei.github.io
==================

# Install

```
bundle install
```

# How to start :

```
bundle exec jekyll serve
# => Now browse to http://localhost:4000
```

If you change somethig in config.yml, redo
```
bundle exec jekyll serve
```


How to make a post :
Copy paste what have been done inside _posts, and then :

```
git push
```
